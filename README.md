# 绿意长存 | EcoGuard 🌿

> “与自然共生，守护人类唯一的蓝色港湾。”

**绿意长存 (EcoGuard)** 是一款专为环境保护公益宣传而设计的现代化、高品质单文件响应式网页。本项目遵循极简、大气的 **Sleek Interface (雅致极简风格)** 与 **Light Glassmorphism (明亮毛玻璃设计)**。项目最大的技术亮点在于：**无外部构建依赖，所有 HTML、CSS 样式和 JavaScript 交互逻辑全自包含于单个 `index.html` 文件中**，实现极速双击运行、零门槛部署与超流畅浏览。

---

## ✨ 核心亮点

### 🎨 卓越的视觉美学
* **明亮轻盈毛玻璃 (Light Glassmorphism)**：基于高透白半透明色调 (`rgba(255, 255, 255, 0.72)`)，搭载细腻的立体透光边框线与柔和的漫反射阴影，营造通透、纯净的高端视觉感受。
* **生机自然渐变背景 (Nature-glowing Blobs Overlay)**：原生 Tailwind CSS 实现的流光溢彩氛围，翡翠绿 (`#22c55e`) 渐变气泡辅以自适应微光，摒弃厚重暗沉。
* **黄金比例极简排版**：精选现代感十足的高端英文字体 `Outfit` 与中文字体 `Noto Sans SC` 优雅对应，页面呼吸感十足，节奏张弛有度。

### ⚡ 核心功能与交互
1. **全动态数据看板 (Dynamic Real-time Dashboard)**：
   * 原生 JS 微任务轮询定时器，实时递增全球环保减数（塑料废弃拦截、联合植树抚育、累计减少碳排总量），带给访问者极强的现场感。
2. **多终端响应式浮动导航栏 (Responsive Hybrid Navbar)**：
   * 桌面端提供高光毛玻璃滑动卡片；移动端自适应折叠为精致的悬浮折叠框。伴随页面滚动，导航栏自适应微调背景模糊度并收紧布局。
3. **低碳减排每日计算器 (Interactive Carbon Calculator)**：
   * 趣味十足的低碳科普工具。访问者勾选其日常低碳举措（替代自驾、拒绝一次性塑料、随手节水、践行光盘行动），即可通过系数矩阵实时计量减排度，全自动激活可爱的成就勋章动感提示。
4. **近期一线下乡动态报告 (Frontline Initiatives Tracker)**：
   * 三维精细卡片完美收纳红树林复绿、海岸清扫、防沙治沙等主力NGO项目，配备精美的彩色状态徽章和原生 CSS 圆角进度条。
5. **极速志愿申报系统 (Interactive SignUp Sandbox)**：
   * 配备个人隐私安全指引、意向领域多选与留言框。内置字段正则深度验证，单击提交后展示流畅的“正在提交中...”加载动效，并在一秒内无缝唤醒温润舒缓的“提交成功”卡片。

## 🛠️ 技术栈与全包含（Single-file）声明

项目极致专注于低代码与绿色部署体验：
* **核心骨架**：HTML5 (语义化纯净标记)
* **样式引擎**：Tailwind CSS (通过官方高性能 CDN 独立加载运行)
* **设计图标**：Font Awesome v6 (高清晰矢量 CDN 托管)
* **动态交互**：Vanilla JS (原生高能 JavaScript)

## 🚀 极速启动与部署

### 方式一：独立双击运行
1. 下载或克隆本项目。
2. 双击 `index.html` 即可立即在 Chrome, Safari 或 Edge 等任何主流浏览器中，体验 100% 完整功能与无瑕动画，无需安装任何额外 Node.js 包环境。

### 方式二：极速零配置服务化
若希望在局域网或个人服务器中部署：
```bash
# 方式 A: 利用 python 快速托管
python3 -m http.server 3000

# 方式 B: 进入浏览器，直接拖入静态云托管服务 (如 GitHub Pages, Vercel 或 Netlify) 即可 3 秒上线
```
